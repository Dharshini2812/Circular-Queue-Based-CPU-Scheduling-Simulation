Round-Robin (Circular Queue) Scheduling

A CPU scheduling algorithm where each process gets a fixed time quantum. If not finished, it returns to the queue.

How to use:

  1. Add processes (arrival time, burst time)
  2. Set quantum (time slice)
  3. Click Start to simulate
  4. Use Play/Pause/Reset to control animation

Metrics:

  • TAT: Completion Time - Arrival Time
  • WT: TAT - Burst Time
